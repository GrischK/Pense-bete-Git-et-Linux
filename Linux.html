Révisions des principales notions de Linux

Pour ce rappel, j'ai utilisé la ressource GitIt.

1.Démarrer avec Linux

Le login

Il faut d'abord se logger:
mateo21-desktop login:

mateo21-desktop : c'est le nom qu'on a donné à notre ordinateur lors de l'installation. Notre ordinateur se présente et nous rappelle où nous sommes, en quelque sorte.

Entrez le login: dans mon cas GrischK

On nous demande ensuite notre mot de passe.

2.Entrer une commande

L'invite de commandes

On travaille dans la console en tapant ce qu'on appelle des commandes.
Les paramètres sont des options que l'on écrit à la suite de la commande. La commande et les paramètres sont séparés par un espace.
Certains paramètres nécessitent que vous les complétiez avec une valeur.

Autocomplétion de commande

Le premier « truc » à connaître, c'est l'autocomplétion de commande. Prenons la commande date par exemple.
Taper juste « da » dans la console, puis tapez deux fois sur la touche Tabulation

L'historique des commandes

Appuyer sur la flèche directionnelle haut. On verra apparaître la dernière commande qu'on a tapée.
Si on appuie sur la flèche directionnelle bas, on reviendra aux commandes les plus récentes.

Les différentes commandes et paramètres

Les commandes :

  • pwd : afficher le dossier actuel
  • ls : lister les fichiers et dossiers
  • cd : changer de dossier
  • du : taille des dossiers
  • cat : afficher tout le fichier
  • less : afficher les fichiers page par page
  • head : afficher le début du fichier
  • tail : afficher la fin du fichier
  • touch : créer un fichier
  • mkdir : créer un dossier
  • cp : copier un fichier
    (ex: cp fichierbidon fichiercopie = "fichiercopie" est le nouveau nom
    cp fichierbidon mondossier/ = fichierbidon sera copié dans mondossier sous le même nom
    cp fichierbidon mondossier/fichiercopie = fichierbidon sera copié dans mondossier sous le nom fichiercopie
    cp -R animaux = avec attribut -R cela copie le dossier animaux)
  • mv : déplacer un fichier / renommer un fichier
    (ex: mv animaux/ mondossier/ = déplacera le dossieranimaux dans mondossier
    mv fichierbidon superfichier = renommera fichierbidon en superfichier
    mv fichierbidon mondossier/superfichier = déplacera fichierbidon dans mondossier tout en lui affectant un nouveau nom)
  • rm : supprimer des fichiers et dossiers
  • ln : créer des liens

Les paramètres :

  • -a (all) tout sélectionner
  • -n affiche les numéros de ligne
  • -f suivre évolutions et changements d'un fichier
  • -r -R dossier
  • * joker ou wildcard
    (ex: cp *.jpg mondossier/ = copie tous les.jpg dans un sous-dossier
    cp so* mondossier/ = copie tous les fichiers commençant par so)
  • -i demande une confirmation
  • -v demande à la commande de dire ce qu'elle est en train de faire
